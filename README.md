### 주요 인센티브 알고리즘

##### Constant Incentvie

```
폰 사용 절제 미션을 성공하면 매 시간마다 500 Gold를 받습니다.
```

##### Random Incentive

```
폰 사용 절제 미션을 성공하면 매 시간마다 200 ~ 800 사이의 임의의 Gold를 받습니다.
```

##### MAB Incentive

```
폰 사용 절제 미션을 성공하면 매 시간마다 Context 기반 200 ~ 800 사이의 Gold를 받습니다.
```
